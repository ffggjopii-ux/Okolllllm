# موقع نشر المحتوى مع نظام التعليقات

موقع بسيط لنشر المحتوى مع نظام تعليقات يتطلب موافقة المشرف.

## المميزات

### للزوار:
- عرض المنشورات
- إضافة تعليقات جديدة
- عرض التعليقات المعتمدة فقط

### للمشرف:
- إدارة المنشورات (إضافة/تعديل/حذف)
- مراجعة التعليقات والموافقة عليها
- حذف التعليقات غير المناسبة

## كيفية الاستخدام

### التشغيل المحلي:
1. انسخ الملفات إلى مجلد على جهازك
2. افتح `index.html` في المتصفح

### النشر على Netlify:
1. ارفع الملفات إلى مستودع GitHub
2. سجل الدخول إلى Netlify
3. اختر "New site from Git"
4. اختر مستودعك
5. اترك الإعدادات الافتراضية وانقر على "Deploy"

## بنية الملفات
- `index.html` - الصفحة الرئيسية للزوار
- `admin.html` - لوحة تحكم المشرف
- `style.css` - ملف التنسيقات
- `script.js` - ملف JavaScript الرئيسي
- `data/` - مجلد يحتوي على بيانات الموقع

## ملاحظات
- البيانات تحفظ في localStorage للمتصفح
- لاستخدام قاعدة بيانات حقيقية، يمكنك استبدال وظائف localStorage بـ API
- الموقع متوافق مع جميع الأجهزة

مشروع مفتوح المصدر للاستخدام التعليمي
